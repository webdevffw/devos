## Ghostty

* **Purpose:** Modern GPU-accelerated terminal
* **Config:** Declarative config file
* **Strengths:** Simplicity, performance, clean defaults
* **Risks:** Newer ecosystem, smaller community

---

## WezTerm

* **Purpose:** Highly configurable terminal emulator
* **Config:** Lua (extremely powerful)
* **Strengths:** Cross-platform, scripting, multiplexing-lite
* **Risks:** Complexity creep
